# ATM Simulator

> **Учебный проект** — создан в образовательных целях для практики Java. Не предназначен для реального использования.

![Java](https://img.shields.io/badge/Java-ED8B00?style=flat&logo=openjdk&logoColor=white)
![CLI](https://img.shields.io/badge/CLI-консоль-blue)

Консольная симуляция банкомата. Пользователь вводит текстовые команды и взаимодействует с виртуальным счётом.

## Возможности

- Пополнение баланса
- Снятие средств (с проверкой на достаточность)
- Просмотр текущего баланса
- Команда `help` для справки
- Выход командой `exit`

## Структура проекта
ATM-Simulator/
├── Main.java       # Точка входа, цикл обработки команд
└── ATM_API.java    # Логика банкомата (баланс, операции)

## Запуск

```bash
javac Main.java ATM_API.java
java Main
```

## Доступные команды

| Команда               | Описание             |
|-----------------------|----------------------|
| `replenish`           | Пополнить баланс     |
| `take off`            | Снять средства       |
| `find out the balance`| Узнать баланс        |
| `help`                | Список команд        |
| `exit`                | Выйти из программы   |

## Что изучалось

- Разделение логики по классам (`Main` и `ATM_API`)
- Статические методы и переменные
- Работа со `Scanner` для ввода с консоли
- Конструкция `switch` для обработки команд
- Простая проверка условий (достаточность средств)
